# RetailPulse — Sales Warehouse & KPI Dashboard

## Overview
RetailPulse is a beginner-friendly data engineering project demonstrating a full ETL workflow:
CSV → Python → PostgreSQL → SQL KPIs → Power BI Dashboard.

## Tech Stack
- Python (Pandas)
- PostgreSQL
- SQL
- Power BI

## ETL Workflow
1. Extract data from CSV
2. Clean and transform using Pandas
3. Load into PostgreSQL
4. Generate KPIs using SQL
5. Visualize in Power BI

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Order Value
- Sales by Category & Region
- Monthly Sales Trend

## How to Run
1. Install requirements
2. Configure database in `config.py`
3. Run `etl.py`
4. Connect Power BI to PostgreSQL
